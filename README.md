# noor to shop

Modern, minimal ecommerce experience focused on warm aesthetics and curated everyday essentials.

## Tech Stack
- React 19 + Vite 7
- Tailwind CSS (v4) with custom Noor theme tokens
- React Router for navigation
- ESLint (flat config)

## Getting Started
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Scripts
| Command | Purpose |
|---------|---------|
| `npm run dev` | Start local dev server |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |

## Structure
```

	components/   # Reusable UI pieces
	pages/        # Routed views
	data/         # Static product data
```

## Branding
Public brand: "noor to shop" (with spaces). Internal package name uses hyphens.
